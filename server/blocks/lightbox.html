Lightbox

Pop up a beautiful full screen responsive lightbox customised for each device

Lightbox Slideshow

Pops up a slideshow with thumbnails for a group of images

Code : Put your pictures inside an element with an id of lightbox. 

#lightbox
  a(href='/pictures/products/product-1.jpg', title='Lamps')
    img(src='/pictures/products/product-1-sm.jpg')
  a(href='/pictures/products/product-2.jpg', title='Chair')
    img(src='/pictures/products/product-2-sm.jpg')
