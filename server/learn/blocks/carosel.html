Carosel

Fire the plugin on a "wrapper" with existing content, and it's replaced with an inline slider. Just like the lightbox popup, the slides in this slider are responsive by default. This slider has been set up to show three slides and is infinite: 

 $("#example").tosrus({
    infinite   : true,
    slides     : {
       visible    : 3
    }
 });
